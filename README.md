# DemoApp
This app contains:
1. Login/SignUp with Firebase (with activity stack properly managed when user signs up, logs in or moves from login to sign up or vice versa.)
2. Spinner with easy implementation.
3. Bottom Navigation Bar/Menu
4. Spinner
5. Graph
6. Pie Chart
